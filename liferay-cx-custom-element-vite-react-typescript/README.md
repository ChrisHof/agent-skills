# Liferay Custom Element Client Extension using Vite and React & TypeScript

This [Agent Skill](https://agentskills.io) creates a [Liferay Custom Element Client Extension](https://learn.liferay.com/w/dxp/development/integrating-external-applications/creating-a-basic-custom-element) using _Vite_ as the build tool and _React & TypeScript_ from a predefined template.

Live Development/Hot reload in your Liferay environment is supported when deploying the CX using the `deployDev` Gradle task.

## Prerequisites

- [Liferay](https://www.liferay.com/) 7.4+ and a Liferay Workspace
- [Gradle](https://gradle.org/) (or Maven)
- [Node.js](https://nodejs.org/) 18+
- [yarn](https://yarnpkg.com/) or [npm](https://www.npmjs.com/)
- IDE, CLI or CI/CD pipeline that supports agent skills

## Usage

Check the docs of your IDE, CLI or CI/CD pipeline on how to use agent skills.
